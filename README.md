Toreda's official `types` for organization-wide TypeScript projects.

## Installation

### Install using yarn

```bash
$ yarn add @toreda/types --dev
```

OR

```bash
$ yarn add @toreda/types --D
```

### Install using npm
```bash
$ npm add @toreda/types --save-dev
```
